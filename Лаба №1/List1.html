Список использованных тегов

  1. a - Создаёт гипертекстовые ссылки.
  2. b - Задает полужирное начертание отрывка текста, не придавая акцент или важность выделенному.
  3. body - Представляет тело документа (содержимое, не относящееся к метаданным документа).
  4. br - Перенос текста на новую строку.
  5. caption - Добавляет подпись к таблице.
  6. div - Элемент-контейнер для разделов HTML-документа. Используется для группировки блочных элементов с целью форматирования стилями.
  7. em - Выделяет важные фрагменты текста, отображая их курсивом.
  8. h1-h6 - Создают заголовки шести уровней для связанных с ними разделов.
  9. head - Элемент-контейнер для метаданных HTML-документа.
  10. hr - Горизонтальная линия для тематического разделения параграфов.
  11. html - Корневой элемент HTML-документа. Сообщает браузеру, что это HTML-документ. Является контейнером для всех остальных html-элементов.
  12. i - Выделяет отрывок текста курсивом, не придавая ему дополнительный акцент.
  13. img - Встраивает изображения в HTML-документ с помощью атрибута src, значением которого является адрес встраиваемого изображения.
  14. ins - Выделяет текст подчеркиванием. Применяется для выделения изменений, вносимых в документ.
  15. li - Элемент маркированного или нумерованного списка.
  16. li - Элемент маркированного или нумерованного списка.
  17. mark - Выделяет фрагменты текста, помечая их желтым фоном.
  18. ol - Упорядоченный нумерованный список. Нумерация может быть числовая или алфавитная.
  19. p - Параграфы в тексте.
  20. span - Контейнер для строчных элементов. Можно использовать для форматирования отрывков текста, например, выделения цветом отдельных слов.
  21. sub - Задает подстрочное написание символов, например, индекса элемента в химических формулах.
  22. sup - Задает надстрочное написание символов.
  23. table - Элемент для создания таблицы.
  24. tbody - Определяет тело таблицы.
  25. td - Создает ячейку таблицы.
  26. th - Создает заголовок ячейки таблицы.
  27. thead - Определяет заголовок таблицы.
  28. title - Заголовок HTML-документа, отображаемый в верхней части строки заголовка браузера.
  29. tr - Создает строку таблицы.
  30. u - Выделяет отрывок текста подчёркиванием, без дополнительного акцента.
  31. ul - Создает маркированный список.
