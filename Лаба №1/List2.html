Список использованных CSS-свойств

  1. background - Устанавливает все свойства фона элемента за одну декларацию.
  2. border - Позволяет определить все атрибуты рамки HTML элемента за одну декларацию.
  3. border-collapse - Определяет, какую модель рамки вокруг ячеек таблицы следует использовать.
  4. border-color - Определяет цвет всех четырех сторон рамки HTML элемента.
  5. border-radius - Устанавливает радиус скругления уголков рамки.
  6. color - Устанавливает цвет текста для различных элементов.
  7. cursor - Определяет вид курсора при наведении мышки на элемент.
  8. display - Определяет, как элемент должен быть показан в документе (тип его отображения).
  9. flex - Позволяет указать параметры элемента, чтобы он эффективно заполнял доступное пространство.
  10. float - Определяет, по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с другой стороны.
  11. font-family - Задает семейство шрифта, которое будет использоваться для оформления текста содержимого.
  12. font-size - Определяет размер шрифта.
  13. font-weight - Устанавливает насыщенность или "жирность" начертания шрифта.
  14. gap - Pадаёт отступы между колонками и рядами.
  15. left - Определяет расстояние от левого края элемента до левого края его ближайшего родительского элемента, т.е. расстояние его смещения вправо.
  16. line-height - Устанавливает межстрочный интервал текста, отсчет ведется от базовой линии шрифта.
  17. list-style-image - Устанавливает изображение, которое будет использоваться в качестве маркера элементов списка.
  18. margin - Позволяет устанавливать размер всех отступов элемента за одну декларацию.
  19. opacity - Определяет уровень прозрачности элемента веб-страницы.
  20. outline - Устанавливает атрибуты контура элемента.
  21. outline-color - Определяет цвет контура.
  22. outline-style - Определяет стиль контура.
  23. overflow - Определяет, каким образом будет отображаться содержимое элемента, если оно не помещается в область вывода данного элемента.
  24. padding - Позволяет устанавливать размер всех полей элемента за одну декларацию.
  25. padding-bottom - Это свойство устанавливает размер нижнего поля элемента.
  26. padding-left - Это свойство устанавливает размер левого поля элемента.
  27. padding-right - Это свойство устанавливает размер правого поля элемента.
  28. position - Устанавливает способ позиционирования элемента относительно окна браузера или других объектов на веб-странице.
  29. resize - Свойство resize определяет, может ли пользователь изменять размер элемента.
  30. right - Определяет расстояние от правого края элемента до правого края его ближайшего родительского элемента.
  31. table-layout - Определяет, как браузер должен вычислять высоту и ширину ячеек таблицы, основываясь на ее содержимом.
  32. text-align - Устанавливает горизонтальное выравнивание текста в пределах элемента.
  33. text-decoration - Добавляет оформление текста в виде подчеркивания, перечеркивания, линии над текстом.
  34. text-decoration-line - Добавляет оформление текста в виде подчеркивания, перечеркивания, линии над текстом.
  35. text-emphasis-color - С помощью данного свойства определяется цвет, выделяющий диакритические знаки, которые прилегают к каждому символу.
  36. text-emphasis-style - Свойство устанавливает внешний вид меток выделения.
  37. text-shadow - Добавляет тень к тексту, а также устанавливает её параметры: цвет тени, смещение относительно надписи и радиус размытия.
  38. width - Устанавливает ширину блочных или заменяемых элементов.
  39. word-break - Определяет, как делать перенос на новую строку внутри слов, которые не помещаются в ширину контейнера.
  40. word-spacing - Устанавливает расстояние между словами в пределах элемента.
